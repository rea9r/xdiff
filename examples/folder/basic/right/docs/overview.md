# Overview

xdiff is a small utility for comparing two payloads.
The desktop GUI is now the primary frontend.

## Modes

- text: line-oriented diff with overview ruler
- json: structural diff with semantic view
- folder: recursive folder compare with filters

## Roadmap

- ship overview ruler everywhere
- add quick filter shortcuts
- improve very-deep-tree performance
